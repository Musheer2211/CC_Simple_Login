💡 EXAMPLE: Complete OAuth Flow
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Option 1: Use the example client app
  $ python example_oauth_client.py
  → Visit http://localhost:3000
  → Click login and follow the OAuth flow

Option 2: Manual curl testing
  # 1. Get OAuth server info
  curl http://localhost:8000/oauth/info
  
  # 2. Register OAuth client (browser at /admin/clients)
  
  # 3. Get auth code (open in browser)
  http://localhost:8000/oauth/authorize?client_id=YOUR_ID&redirect_uri=...
  
  # 4. Exchange code for tokens
  curl -X POST http://localhost:8000/oauth/token \
    -d "client_id=YOUR_ID" \
    -d "client_secret=YOUR_SECRET" \
    -d "code=AUTH_CODE" \
    -d "redirect_uri=YOUR_URI" \
    -d "grant_type=authorization_code"
  
  # 5. Use access token to get user info
  curl -H "Authorization: Bearer ACCESS_TOKEN" \
    http://localhost:8000/oauth/userinfo

  GET    /oauth/authorize       Request authorization code
  POST   /oauth/authorize       Approve authorization

  GET    /oauth/userinfo        Get authenticated user information

  POST   /admin/clients         Register new OAuth client
